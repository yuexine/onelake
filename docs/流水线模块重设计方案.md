# 流水线模块当前设计：Spark-only 统一编辑器

> 修订日期：2026-07-09
>
> 本文替换旧版包含 dbt/Trino 主线的重设计文档。当前代码主线是 Spark-only pipeline，前端主入口是 `UnifiedPipelineEditor`。

## 1. 当前事实

| 区域 | 当前实现 |
| --- | --- |
| 后端模块 | `module-orchestration` |
| 主要 Controller | `PipelineController`、`PipelineBackfillController`、`OperatorController`、`JobRunController`、`ResourceGroupController`、`RuntimeContractController`、`DagController` |
| 数据模型 | `dag`、`job_run`、`task_run`、`pipeline_task`、`pipeline_task_edge`、`operator`、`operator_version`、`operator_install`、`resource_group`、`compute_profile` |
| 运行主线 | Spark SQL / PySpark / Quality Gate |
| Dagster job | `onelake_pipeline_run` |
| 前端 | `/orchestration/pipelines/:id` 使用 `UnifiedPipelineEditor`；历史 `/:id/graph` 重定向 |

## 2. 任务类型

当前文档统一写：

```text
SYNC_REF
SPARK_SQL
PYSPARK
QUALITY_GATE
```

旧的 `TRINO_DBT`、dbt model run 或 Trino transform 作为历史方案，不再写入当前主线。

## 3. 编辑器职责

统一编辑器需要表达：

1. 任务节点和数据流边。
2. 输入/输出端口契约。
3. 资源画像。
4. 运行参数。
5. 质量门禁。
6. 校验和编译结果。
7. 触发、回填、运行实例。
